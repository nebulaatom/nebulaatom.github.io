Nebula Atom

High-Performance Web Framework

Nebula Atom is a C++ Web Framework for building high-performance APIs, supporting scalable architecture and modern web technologies.

Download Docs

C++ Web Framework

Introducing Nebula Atom, a high-performance web framework built on top of the POCO libraries. Nebula Atom is designed to address the challenges of modern web development, providing developers with a powerful and versatile toolkit for crafting exceptional web experiences.

Key Features of Nebula Atom

Configurable Server

Allows you to configure the maximum number of threads (that manage requests), queued requests, etc.

Asynchronous / Synchronic programming

allows queries to databases synchronously or asynchronously (MariaDB/MySQL/PostgreSQL/SQLite3).

Functions and Actions System

Provides an automatic system for mapping, composition, execution and delivery of results of database requests.

HTTP Modules

Http/1.1, HTTP Headers and Cookies, HTTPS, WebSocket, etc.

Extra Modules

Routing, JSON, File management, etc.

High Performance

An office laptop with Ryzen 7 can process more than 50,000 HTTP requests per second.

Purpose

In today's demanding web landscape, performance is paramount. Users expect websites to load quickly, be responsive, and handle heavy workloads without compromising user experience. Developers, on the other hand, need tools that empower them to build performant and scalable web applications without sacrificing development speed or flexibility.
